مياه الشرب
نظام إدارة مياه الشرب – تطبيق ويب لإدارة الاشتراكات وقراءات العدادات والفواتير.
الوصف
يوفر هذا المشروع واجهة بسيطة وفعالة لإدارة بيانات المشتركين والفواتير الشهرية، ويتيح:
تسجيل المشتركين وبياناتهم.
متابعة القراءات السابقة والحالية للعدادات.
حساب المبالغ المستحقة والمتأخرات تلقائيًا.
إرسال رسائل ترحيب صوتية عند دخول المستخدم إلى الموقع.
المميزات
إدارة شاملة للاشتراكات والفواتير.
حساب تلقائي للفرق بين القراءة السابقة والحالية.
دعم عرض المبالغ المستحقة والمتأخرات لكل مشترك.
تجربة تفاعلية مع الصوت الترحيبي.
كيفية الاستخدام
افتح الملف index.html في أي متصفح حديث.
يمكن تشغيل الصوت الترحيبي تلقائيًا أو عند الضغط على زر الترحيب.
ملاحظة
المشروع يعتمد على HTML وJavaScript، لذا يفضل استخدام متصفح حديث لدعم كل الميزات.
